# FaBo Blocks Docs For Raspberry pi pico


##FaBo Blocks

&emsp;RJ11ケーブルでセンサーを用いましてRaspberryPico,RaspberryPico Wと簡単接続します。ブレッドボードや半田接続を必要とせず。ブロックで固定でき授業や研修がとてもスムーズになります。
